The importance of fonts in your projects



It is important to know how to choose fonts for our projects. Not all computers have the same fonts installed. If you don't take into account some important processes regarding your fonts, your website will look different from what you want it to. It will look different from what you designed.


For example, if the font you chose is not installed on a computer, will result in your page looking different.


On this page, we are going to investigate web fonts. My goal is to learn the necessary steps to get my websites to display the way I want them to on all browsers and devices. Here you will find some methods and instructions for installing and using fonts. I will add to this page as the research progresses.

About fonts



Typography Basics

There are five basic classifications of typefaces: serif, sans serif, script, monospaced, and display.


  • Serif: Garamond, Baskerville, Perpetua
  • Sans serif: Helvetica, Arial, Verdana
  • Script: Those are less common. E.G. Old English
  • Monospaced: Courier New
  • Display: less used, E.G. Broadway

How to make sure fonts are displayed correctly everywhere



There are several ways to insert your fonts into a web page. Here we will look at some of them:


Google Fonts

Google fonts is very easy to use. Just go to the page, choose some fonts and copy the code that the wizard generates into the head of your HTML page. Of course, each page you want to use these fonts must have the code generated by Google in the head. This tells your page to fetch those fonts from Google's database. This way you make sure that the font you chose will always be displayed correctly in any browser and on any device. You should also copy the CSS rules that the wizard automatically generates for you.


Download font and put it in your project folder using the CSS rule @font-face

You must first search for your font on the web. Once you find the font you want, download it. You should look for a file with an .otf extension, for example, Quicksand.otf. This is your font file. Once you find it, you have to make sure you copy it to your website folder.

Titles in Times New Roman


Times New Roman is a serif typeface commissioned by newspaper The Times (London).

Titles in Palatino


Palatino is the name of an old-style serif typeface designed by Hermann Zapf, initially released in 1949 by the Stempel foundry and later by other companies, most notably the Mergenthaler Linotype Company.

Titles in Impact


Impact is a sans-serif typeface in the industrial or grotesk style designed by Geoffrey Lee in 1965 a It is well known for having been included in the core fonts for the Web package and distributed with Microsoft Windows since Windows 98. In the 2010s, it gained popularity for its use in image macros and other internet memes.[




Titles in Montserrat


Montserrat is a geometric sans-serif typeface designed by Argentine graphic designer Julieta Ulanovsky and released in 2011. It was inspired by posters, signs and painted windows from the first half of the twentieth century, seen in the historic Montserrat neighbourhood of Buenos Aires.

Titles in Courier
